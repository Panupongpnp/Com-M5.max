รางวัลสะสม

( ComCamp 35 )

จัดโดย คณะวิศวกรรมศาสตร์คอมพิวเตอร์ มจธ.

ซึ่งจะคอยให้คำแนะนำในการศึกษาต่อ และกิจกรรมเวิร์คช็อปต่างๆ ให้รู้จักคณะวิศวคอมพิวเตอร์


( JPC16 )
